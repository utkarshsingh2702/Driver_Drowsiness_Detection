# Driver Drowsiness Detection

This project detects driver drowsiness using computer vision techniques
based on eye aspect ratio (EAR) and mouth aspect ratio (MAR).

## Features
- Real-time drowsiness detection using webcam
- Eye blink and prolonged eye closure detection
- Yawn detection
- Audible alert on drowsiness

## Requirements
- Python 3.8+
- Webcam
